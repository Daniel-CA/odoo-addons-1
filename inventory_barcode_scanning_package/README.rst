==============================================
Barcode scanning in inventory v11 with Package
==============================================

Extension of Barcode scanning in inventory v11 to packages.

Depends
=======
[inventory_barcode_scanning] https://apps.odoo.com/apps/modules/11.0/inventory_barcode_scanning/

Usage
=====

You have to activate Product Packaging in inventory config.
